How does monitor404 work?

Whenever a user clicks on a link on your site, the monitor404.js script sends a request to the monitor404 server. monitor404 asyncronously checks the link (by issuing a HEAD request) to determine the the link is valid. monitor404 caches link checks to minimize load on frequently clicked links. The script is completely invisible to the user and optionally configurable to only monitor certain links.

Check out the documentation for more info.
